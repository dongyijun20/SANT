# SANT Statistical Analysis

This repository provides the analytical script used in this study.

## Code

- `analysis.R`: the custom script for data processing, statistical analysis, and figure generation.

## Data and Code Availability

The custom code and analytical scripts used for the statistical analyses in this study are publicly accessible via the GitHub repository: [https://github.com/dongyijun20/SANT](https://github.com/dongyijun20/SANT).  
The clinical and pathologic datasets analyzed during the current study are not publicly available to protect patient privacy and confidentiality, but they are available from the corresponding author upon reasonable request and subject to institutional review board approval.
